
## 项目简介
语音绘图工具的后端服务，提供手绘作品管理、AI语音交互等能力。

## 已完成功能
### 1. 作品管理
- 保存手绘作品（图片 + 操作序列）
- 获取作品列表
- 作品详情（归属校验）
- 逻辑删除
### 2. 用户认证
- 手机号 + 验证码登录
- JWT Token 认证
- 登录拦截器
### 3. AI语音能力（进行中）
- 阿里云ASR语音识别
- 阿里云TTS语音合成
- AI问答
## 技术栈
| 技术 | 说明 |
|------|------|
| Spring Boot 3.2.5 | 后端框架 |
| MyBatis-Plus 3.5.6 | ORM框架 |
| MySQL | 数据库 |
| Redis | 缓存 |
| JWT | 身份认证 |
| 阿里云NLS | 语音识别/合成 |
